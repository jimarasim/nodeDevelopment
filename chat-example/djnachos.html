TUNES

This page does not have a Database, nor does it use Cookies; i.e. No data is stored, kept, or sold.
Videos, Music, and Images are all hosted remotely.
You do not need to use an Alias to send a Chat Message.
If you send a Chat Message that's only an Image URL (e.g. "https://seattlerules.com/whale.jpg"), the image will render in the chat.
If you send a Chat Message that's only an MP3 URL (e.g. "http://analogarchive.com/analogarchive/analog/blackflag-louielouieBWdamagedI.mp3"), the TUNES player will start playing that song.
If you send a Chat Message that's only an MP4 URL (e.g. "http://www.helena.seattlerules.com/media/jim/IMG_3273.mp4"), the Video Player will start playing that video.
